   - Props: `{ expanded: boolean, onToggle: () => void, onRemove: () => void, summary: string, children: React.ReactNode }`
   - Uses shadcn Card, CardHeader, CardContent components
   - CardHeader is clickable (cursor-pointer) to toggle expand/collapse
   - Shows summary text on left (or "New Entry" if empty), chevron icon + trash icon on right

   - Trash icon is a ghost Button with size="icon", calls onRemove with e.stopPropagation() to prevent toggle
   - CardContent renders children only when expanded
   - Import ChevronDown and Trash2 from lucide-react

2. Create TagInput component (src/components/shared/tag-input.tsx):
   - 'use client' directive
   - Props: `{ value: string[], onChange: (tags: string[]) => void, placeholder?: string }`
   - Renders a bordered container with flex-wrap layout
   - Each tag is a shadcn Badge (variant="secondary") with an X button to remove
   - At the end, a plain `<input>` (no border, transparent background) for typing
   - On Enter: if input has content, add trimmed value to tags array via onChange, clear input. Prevent form submission with e.preventDefault()
   - On Backspace: if input is empty and tags exist, remove last tag
   - Do NOT add duplicate tags — check if tag already exists before adding
   - Clicking the container focuses the input (useRef)
