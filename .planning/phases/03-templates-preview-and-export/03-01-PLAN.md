   - `SectionTitle`: View + Text with consistent heading style
   - `DateRange`: Text showing formatted date range (handles "Present" for missing endDate)
   - `BulletList`: View rendering an array of strings as bullet points
   - `ContactLine`: Text showing email | phone | location | url in a row
   - All use @react-pdf/renderer primitives (View, Text, StyleSheet) — NOT HTML elements

6. Create `src/components/templates/classic-template.tsx`:
   - Import Document, Page, View, Text, StyleSheet from @react-pdf/renderer
   - Import shared primitives and fonts
   - Props: `{ resume: ResumeData }` — pure renderer, NO hooks, NO state
   - Page size A4, 40pt padding, Inter font family
   - Style: Sans-serif, subtle gray accents (#4A5568), generous whitespace per CONTEXT.md
   - Sections in order: Header (name, label, contact), Summary, Experience, Education, Skills, then optional sections (certificates, projects, languages, volunteer) only if they exist (check array is defined and non-empty)
   - Experience entries: company name bold, position + dates, bullet points from highlights array
   - Education entries: institution bold, degree/area, dates, GPA if present
   - Skills: grouped by category name with keywords as comma-separated list
   - Use flexbox layout (NOT CSS Grid — not supported by @react-pdf/renderer)
   - Handle empty/missing data gracefully (don't render empty sections)

2. Create `src/components/export/pdf-export.ts`:
   - Import pdf from @react-pdf/renderer
   - Import saveAs from file-saver
   - Import TEMPLATES from template-registry
   - Export async function `exportPdf(resume: ResumeData, templateId: TemplateId)`:
     - Get template component from TEMPLATES[templateId]
     - Create JSX: `<TemplateComponent resume={resume} />`
     - Call `pdf(jsx).toBlob()` to generate PDF blob
     - Call `saveAs(blob, filename)` where filename = `${resume.basics.name || 'resume'}.pdf`
   - This function uses the SAME template component as the preview panel — guaranteeing PREV-02 (preview matches export)
